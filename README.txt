# lyon_equipements.csv

This repo contains data from Lyon City's official website.

The table has equipments as individuals, and the following columns:

    titres : Name of the equipment
    adresses: Postal address of the equipment
    liens: urls to equipment description
    categorie: categories of equipment, as defined on Lyon City's website
    type: broader types (refactored from categorie) of equipment
    lon: longitude
    lat: latitude

This data has been collected on October 22nd, 2019.
